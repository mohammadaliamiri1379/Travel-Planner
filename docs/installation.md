# Installation

## Stable release

To install Travel_planner, run this command in your terminal:

```sh
uv add travel_planner
```

Or if you prefer to use `pip`:

```sh
pip install travel_planner
```

## From source

The source files for Travel_planner can be downloaded from the [Github repo](https://github.com/Mohammadali Amiri/travel_planner).

You can either clone the public repository:

```sh
git clone https://github.com/Mohammadali Amiri/travel_planner
```

Or download the [tarball](https://github.com/Mohammadali Amiri/travel_planner/tarball/main):

```sh
curl -OJL https://github.com/Mohammadali Amiri/travel_planner/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd travel_planner
uv sync
```
